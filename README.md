## Country Detail App

Get country details in one go :zap:

## Usage

Search for any country names on the search bar to get its details.

## Setting up Locally

Get your API keys from <link>

Add the key to the `.env` file:

```bash
cp .env.example .env
```

Install dependencies :star2:

```bash
npm install
```

Run the server :rocket:

```bash
npm start
```
